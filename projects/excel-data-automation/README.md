# Data Automation Project (MS Excel)

## Overview
An Excel-based automation project focused on streamlining employee attendance reporting. The project demonstrates practical automation skills using **Power Query** for data transformation and **VBA Macros** for automated data entry — reducing manual effort in recurring attendance reporting processes.

## Project Scope
- **Data Validation** — Generated and validated 50 rows of employee attendance data in the `Form_Entry` worksheet, ensuring clean and consistent input data as the foundation for automation.
- **Power Query** — Built a Power Query workflow to automatically calculate employee work duration from check-in and check-out timestamps, eliminating manual calculation.
- **Macros (VBA)** — Developed a macro to automate the data entry process, allowing new attendance records to be input quickly and consistently without repetitive manual steps.

## Tools & Skills
| Tool | Purpose |
|---|---|
| Excel – Form_Entry Worksheet | Raw data validation & structuring |
| Power Query | Work duration calculation & data transformation |
| VBA Macro | Automated data input |

## Folder Contents
- `dataset.xlsx` — raw employee attendance dataset
- `attendance_automation.xlsm` — final macro-enabled workbook with Power Query transformation and automated data entry

## Key Takeaway
This project showcases the ability to design an end-to-end Excel automation solution — from raw data validation, to transformation logic with Power Query, to process automation with Macros — directly addressing a common operational need: reducing manual time spent on recurring attendance reporting.
